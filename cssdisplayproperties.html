CSS Display Properties

1. display: block; Elementos com display: block; ocupam toda a largura disponível e começam em uma nova linha. Exemplos de elementos de bloco são div, h1, p, e section.

Elemento de bloco 2

2. display: inline; Elementos com display: inline; ocupam apenas o espaço necessário e não iniciam uma nova linha. Exemplos de elementos inline são span, a, strong, e em.

Elemento inline 1 Elemento inline 2

3. display: inline-block; Elementos com display: inline-block; combinam as características de elementos inline e block. Eles não iniciam uma nova linha, mas permitem definir largura e altura.

Inline-Block 2

4. display: none; Elementos com display: none; são removidos do fluxo do documento, ou seja, não são exibidos na página.

5. display: flex; Elementos com display: flex; ativam o modelo de layout flexível, permitindo uma organização mais eficiente dos itens no container. É útil para criar layouts responsivos.
